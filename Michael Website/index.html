Project Warehouse

In this project, I was grouped into a team of 5, and we were tasked a made-up warehouse by our teachers, we had to create an STP and find bugs or issues and report them using a program named TestRail, it had a free trail so that was perfect for us to use for this one, we were also given the product details of that warehouse, it was supposed to replace an old system that included using Microsoft Excel and handwritten information. The new system we got in the product details contained all of the information including a detailed explanation about the users of the system, a technical overview of the majority of screens in the software, the system also has BATCH processes and Query option that connect to a database.

At the end of the project, we handed our STP projects and presented the presentation in front of an audience, I've included a copy of the document and the presentation

Project Web

In that project, my teachers tasked us to find a website that we need to find bugs or issues in and report them using a project management program named Jira, then they divided us into groups of 2, so after I found my partner we started to look for a buggy website.
we stumbled upon a school website that offered a promotional and school media website that didn't exactly offer the best experience, it also had some serious security issues, since everyone can access the website and browse, download and use the photos of the children that learn in that school, without any login.
After we finished checking the website in two circles, one using a pre-made checklist and the second by exploratory testing, we reported all the issued and bugs into Jira, made an STR document, and then presented it in a presentation
